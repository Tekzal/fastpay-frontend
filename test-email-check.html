Email Check Test

This is a simple test to verify the email check functionality.

Open your browser console and go to the signup page to test the email check.

Expected Behavior:

  • Type an email in Step 2 of the signup form
  • Should see console logs showing the API request
  • Should see visual feedback (green/red border, status message)
  • For "clement.quoda@gmail.com" should show "Email is already taken"
  • For new emails should show "Email is available"

Common Issues:

  • 405 Error: Backend doesn't accept GET requests
  • 404 Error: Endpoint doesn't exist
  • 422 Error: Wrong request format
  • Network Error: Backend server not running

Debug Steps:

  1. Check browser console for error messages
  2. Verify backend server is running
  3. Test endpoint directly: GET /auth/check-email?email=test@test.com
  4. Check API base URL configuration
